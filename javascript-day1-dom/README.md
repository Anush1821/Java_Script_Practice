# JavaScript Day 1 - DOM Manipulation

## 📌 Project
Student Registration application built using Vanilla JavaScript.

## 🚀 Features
- Add a student
- Form validation
- Dynamic DOM manipulation
- Delete student
- Event handling using `submit`
- Prevent page refresh using `preventDefault()`

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript (DOM)

## 📚 Concepts Learned
- createElement()
- appendChild()
- textContent
- addEventListener()
- preventDefault()
- remove()
- DOM Manipulation
